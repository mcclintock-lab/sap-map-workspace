# Publications

This module is an implementation of previously published work.  The Spatial Access Priority Mapping (SAPM) framework has been found to be a good general purpose implementation, having the flexibility to support multiple use cases and scale up to larger populations.

Yates KL, Schoeman DS (2013) Spatial Access Priority Mapping (SAPM) with Fishers: A Quantitative GIS Method for Participatory Planning. PLoS ONE 8(7): e68424. [https://doi.org/10.1371/journal.pone.0068424](https://doi.org/10.1371/journal.pone.0068424)
